# Learning Log

Every research-loop run should leave a short note here when it changes what we
believe.

Use this template:

```markdown
# YYYY-MM-DD - Short Title

## Milestone

M1-real-worker-pool-evaluation

## Question

What did we try to learn?

## Method

What command, dataset, worker pool, or code path did we use?

## Result

What happened? Include numbers.

## Interpretation

What do we believe now?

## Next Step

What should the next run do?
```
